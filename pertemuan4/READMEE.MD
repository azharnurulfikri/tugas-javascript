# 🚗 Sistem Rental Mobil Sederhana

Tugas eksperimen JavaScript menggunakan konsep **OOP (Object Oriented Programming)** dan dijalankan menggunakan **Node.js**.

## 📁 Isi Folder
- `main.js`: File utama yang berisi Class `Pelanggan` dan logika untuk menambah serta menampilkan daftar penyewa.

## 🛠️ Konsep yang Digunakan
1. **Class & Constructor**: Untuk membuat blueprint data pelanggan.
2. **Array**: Sebagai tempat penyimpanan (database) sementara.
3. **Template Literals**: Menggunakan backtick (`` ` ``) untuk output pesan yang dinamis.
4. **Console Table**: Menampilkan data dalam bentuk tabel di terminal agar lebih rapi.

## 🚀 Cara Menjalankan
Pastikan kamu sudah menginstal Node.js, kemudian jalankan perintah berikut di terminal:

```bash
node main.js
