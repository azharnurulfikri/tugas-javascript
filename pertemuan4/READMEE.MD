# 📚 Tugas Pertemuan 4: Sistem Manajemen Transportasi

Repository ini berisi penyelesaian Tugas Pertemuan 4 mata kuliah Pemrograman, yang berfokus pada implementasi **Object-Oriented Programming (OOP)** menggunakan JavaScript.

## 📝 Deskripsi Tugas
Membuat sistem manajemen transportasi sederhana untuk mencatat data pelanggan yang menyewa kendaraan dengan kriteria sebagai berikut:
1.  **Class Pelanggan**: Memiliki properti `nama`, `nomorTelepon`, dan `kendaraanDisewa`.
2.  **Metode Transaksi**: Menambahkan fungsi untuk mencatat aktivitas penyewaan.
3.  **Display System**: Menampilkan daftar pelanggan aktif dalam format yang mudah dibaca.

## 🛠️ Implementasi Kode (OOP Concept)
Dalam project ini, saya menerapkan beberapa konsep inti:
- **Encapsulation**: Data pelanggan dibungkus dalam satu Class.
- **Method**: Fungsi `catatTransaksi()` untuk memproses logika bisnis di dalam objek.
- **Data Collection**: Menggunakan Array untuk menampung seluruh objek pelanggan.
