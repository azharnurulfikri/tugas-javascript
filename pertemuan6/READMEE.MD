# Tugas Manipulasi Data - ES6 Modules

Tugas ini merupakan bagian dari kurikulum **SIB Mandiri Fullstack Web Development di NF Academy**. Proyek ini mendemonstrasikan penggunaan ES6 Modules (`import/export`) dan manipulasi data array of objects di JavaScript.

## 📁 Struktur File
- `data.js`: Berisi data awal (10 user) dalam bentuk Array of Object yang di-export.
- `controller.js`: Berisi logika utama untuk melihat, menambah, dan menghapus data menggunakan method ES6 seperti `.map()`, `.push()`, dan `.splice()`.
- `package.json`: Konfigurasi proyek Node.js untuk mengaktifkan fitur ES6 Modules.

## 🚀 Fitur & Logika
1. **Melihat Data**: Menampilkan seluruh data user ke konsol menggunakan method `.map()`.
2. **Menambah Data**: Menambahkan minimal 2 data baru ke dalam array menggunakan `.push()`.
3. **Menghapus Data**: Menghapus data berdasarkan index tertentu menggunakan `.splice()`.
