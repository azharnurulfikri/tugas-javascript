# 🛒 Sistem Manajemen Inventaris - Pertemuan 3

Tugas ini merupakan implementasi sistem manajemen produk sederhana menggunakan JavaScript, HTML, dan CSS. Proyek ini mendemonstrasikan pengolahan data menggunakan Array of Objects dan manipulasi DOM untuk menampilkan data secara dinamis ke dalam tabel.

## 🚀 Fitur Utama
- **Tampilan Tabel Otomatis**: Data produk dari array ditampilkan langsung di halaman web.
- **Manajemen Produk**: Bisa menambah produk baru melalui input prompt dan menghapus produk berdasarkan ID.
- **UI Aesthetic**: Menggunakan efek Glassmorphism (putih transparan) agar tetap selaras dengan background.
- **Responsif**: Layout yang tetap rapi di berbagai ukuran layar.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur halaman dan tabel.
- **CSS3**: Styling, background image (cover), dan efek blur.
- **JavaScript (ES6)**: 
  - Array of Objects untuk penyimpanan data.
  - Fungsi `push()` untuk menambah data.
  - Fungsi `filter()` untuk menghapus data.
  - `toLocaleString()` untuk format mata uang Rupiah.

## 📂 Cara Menjalankan
1. Pastikan file `index.html` dan `background.jpg` berada dalam satu folder.
2. Buka file `index.html` menggunakan browser pilihan Anda.
